    If automated script doesn't work:

    Step 1: Create hooks folder
        mkdir sahayak-admin\src\hooks

    Step 2: Copy NEW files
        sahayak-pages-Login.jsx
          → src/pages/Login.jsx

        sahayak-hooks-useAuth.jsx
          → src/hooks/useAuth.jsx

        sahayak-components-ProtectedRoute.jsx
          → src/components/ProtectedRoute.jsx

    Step 3: Replace UPDATED files
        sahayak-src-App-updated.jsx
          → src/App.jsx

        sahayak-components-Sidebar-updated.jsx
          → src/components/Layout/Sidebar.jsx

        sahayak-components-Header-updated.jsx
          → src/components/Layout/Header.jsx

    Step 4: Run the app
        cd sahayak-admin
        npm run dev
